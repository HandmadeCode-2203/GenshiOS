# GenshiOS [WIP]

まず初めに、このOSはテスト段階（WIP）です。
RAMがわずか2KBしかないArduino Nano（ATmega328P）等の極限環境で動作することを目指して開発しています。

 #現在搭載している機能#
- CLI操作**：シリアルモニターを通じた対話型インターフェース
- システムメモ帳**：テキストの作成・閲覧履歴の確認
- 簡易電卓**：四則演算機能
- 軽量ファイルシステム**：作成したデータの保存・管理（`.txt` / `.gns` 拡張子対応）
- 独自スクリプト実行エンジン**：独自のプログラミング言語（GenshiScript）をその場でパースして実行し、ハードウェア（LED等）を制御可能

---

## 開発状況について
機能がほとんど載っていない実験段階のシステムですが、2KBのメモリの限界に挑みながら少しずつ育てています。テスト中ですので、温かい目で見守っていただけると幸いです！
